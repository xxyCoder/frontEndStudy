# 运算符的扩展
- 指数 **
  - 特点是右结合的
- 链判断运算符
  - ?.，当前面值存在的时候才会使用.取后面的值
  - obj?.prop // 对象属性是否存在
  - obj?.[expr] // 同上
  - func?.(...args) // 函数或对象方法是否存在
  - 如果属性链有圆括号，链判断运算符对圆括号外没有影响
    - (a?.b).c  === (a === null ? undefined : a.b).c
- Null判断运算符
  - ??，只有左侧值为null | undefined的时候，才会返回右侧的值
- #! 命令
  - 放在脚本的第一行，用于脚本的执行器
  - 对于JS引擎来说，会当成注释
